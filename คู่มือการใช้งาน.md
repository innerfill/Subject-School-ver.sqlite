# คู่มือการใช้งาน ระบบจัดตารางเรียน

---

## การติดตั้ง (ครั้งแรกครั้งเดียว)

### สิ่งที่ต้องเตรียม
- Windows 10 หรือ 11
- ไฟล์ folder โปรแกรมทั้งหมด

### ขั้นตอน

**1. ติดตั้งโปรแกรม**
- ดับเบิลคลิก `install.bat`
- รอจนติดตั้งเสร็จ (ใช้เวลา 5–10 นาที ขึ้นอยู่กับ internet)

**2. เปิด XAMPP และสร้างฐานข้อมูล**
- เปิด XAMPP Control Panel
- กด **Start** ที่ช่อง MySQL
- เปิด browser ไปที่ `http://localhost/phpmyadmin`
- คลิก **New** ทางซ้าย → ตั้งชื่อ `school_schedule` → กด **Create**
- คลิกที่ชื่อ `school_schedule` → เลือกแท็บ **Import**
- กด **Choose File** → เลือกไฟล์ `school_schedule.sql` → กด **Import**

**3. เริ่มใช้งาน**
- ดับเบิลคลิก `start.bat`
- เปิด browser ไปที่ `http://localhost:3000`

---

## การเปิดใช้งานทุกครั้ง

1. เปิด XAMPP Control Panel → กด **Start** ที่ MySQL
2. ดับเบิลคลิก `start.bat`
3. เปิด browser → `http://localhost:3000`

> ห้ามปิดหน้าต่างดำ (Command Prompt) ขณะใช้งาน — ถ้าปิดระบบจะหยุดทำงาน

## เมนูหลัก

### ข้อมูลพื้นฐาน
ต้องกรอกข้อมูลนี้ก่อนจัดตาราง ตามลำดับ:

| เมนู | ใช้สำหรับ |
|------|----------|
| ปีการศึกษา | กำหนดเทอมที่ใช้งาน (ต้องมี 1 เทอมที่ Active) |
| ข้อมูลทั่วไป | ระดับชั้น / แผนก / อาคาร |
| ข้อมูลครู | เพิ่ม/แก้ไขข้อมูลครู |
| รายวิชา | เพิ่มวิชา กำหนดจำนวนคาบ/สัปดาห์ |
| ห้องเรียน | เพิ่มห้องและความจุ |
| ชั้นเรียน | เพิ่มห้องเรียน เช่น ม.1/1 |
| คาบเรียน | กำหนดเวลาแต่ละคาบ |

### จัดการการสอน

| เมนู | ใช้สำหรับ |
|------|----------|
| มอบหมายงานสอน | กำหนดว่าครูคนไหนสอนวิชาอะไร ห้องไหน |
| ล็อกตาราง/กิจกรรม | ล็อกคาบกิจกรรมที่ตายตัว เช่น ลูกเสือ ชุมนุม |
| ภาระงานสอนครู | ดูจำนวนคาบสอนของครูแต่ละคน |
| ตรวจสอบโครงสร้างเวลาเรียน | ตรวจชั่วโมงเรียนตามหลักสูตร |
| ตรวจตารางชน | ตรวจหาปัญหาในตาราง เช่น ครูสอนซ้อน วิชาซ้ำ |

### จัดตาราง
หน้าหลักสำหรับจัดตารางเรียน:
- **ลากวาง** วิชาจาก sidebar ซ้ายไปวางในตาราง
- **ลากย้าย** คาบที่จัดแล้วไปช่องอื่น
- คาบที่มีกุญแจ 🔒 = ล็อกแก้ไขไม่ได้
- ระบบแจ้งเตือนทันทีถ้าตารางชนกัน

### ออกรายงาน
- **ตารางเรียนรายห้อง** — พิมพ์ตารางให้นักเรียน
- **ตารางสอนรายครู** — พิมพ์ตารางให้ครู
- **โครงสร้างรายวิชา** — รายงานวิชาและครูผู้สอน

### จัดสอนแทน
- บันทึกครูลา
- เลือกครูสอนแทน
- แจ้งเตือนผ่าน LINE

## ตรวจตารางชน

ระบบตรวจสอบ 4 ประเภท:

| ประเภท | ความหมาย |
|--------|----------|
| โอเวอร์โหลดรายสัปดาห์ | ครูสอนเกิน 25 คาบ/สัปดาห์ |
| โอเวอร์โหลดรายวัน | ครูสอนเกิน 6 คาบ/วัน |
| สอนต่อเนื่องเกิน | ครูสอนติดกันเกิน 3 คาบ |
| วิชาซ้ำในวันเดียว | วิชาเดิมอยู่ในตารางวันเดียวกันมากกว่า 1 คาบ |

---

## การสำรองข้อมูล

แนะนำ backup ข้อมูลทุกสัปดาห์:
- เข้าเมนู **ตั้งค่า** → แท็บ Backup
- กด **สำรองข้อมูล** → บันทึกไฟล์ `.sql` เก็บไว้

---

## ปัญหาที่พบบ่อย

| อาการ | วิธีแก้ |
|-------|--------|
| เปิด `start.bat` แล้วขึ้น MySQL ยังไม่รัน | เปิด XAMPP → กด Start ที่ MySQL ก่อน |
| เข้า `localhost:3000` ไม่ได้ | ตรวจว่า `start.bat` ยังเปิดอยู่ไหม |
| ข้อมูลหาย | Restore จากไฟล์ backup ใน phpMyAdmin |
| ตารางชนแต่ลากวางได้ | ระบบป้องกันการชนอัตโนมัติ — ถ้าชนจะแจ้งเตือนและไม่บันทึก |
